Project Features

Status Label

If you click on 'Pending', it'll mark the item as 'Done'. Clicking it again will change it back to 'Pending'. If you drag and drop an item into the various lists, it'll automatically mark itself according to where it is. Try it out!

Drag and Drop functionality

Drag and drop works properly with mouse clicks. I've tried to add in touch screen drag and drop capability myself without a library. At the moment, it mostly works, just you can't see the list item as you're dragging.

Storage

At the moment, if you reload the browser you'll lose your to do list - so don't refresh or exit the browser unless you want that to happen! Working on changing this, watch this space!
